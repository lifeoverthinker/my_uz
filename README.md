\# MyUZ Android 🎓



Mobilna aplikacja studencka stworzona, aby ułatwić zarządzanie życiem uniwersyteckim. Aplikacja pozwala na śledzenie planu zajęć, ocen, nieobecności oraz zadań domowych.



!\[Android Build](https://github.com/TWÓJ\_NICK/NAZWA\_REPOZYTORIUM/actions/workflows/android\_build.yml/badge.svg)



\## ✨ Funkcjonalności



Aplikacja oferuje szereg funkcji wspierających studenta:



\* \*\*📅 Plan Zajęć (Kalendarz):\*\* Przejrzysty widok harmonogramu zajęć.

\* \*\*📊 Oceny:\*\* Śledzenie postępów w nauce, obliczanie średnich, szczegóły przedmiotów.

\* \*\*🚫 Nieobecności:\*\* Monitorowanie frekwencji na zajęciach.

\* \*\*📝 Zadania:\*\* Lista zadań do wykonania (To-Do) zintegrowana z kalendarzem.

\* \*\*⚙️ Personalizacja:\*\* Ustawienia motywu, edycja danych studenta.



\## 🛠️ Technologie



Projekt został napisany w nowoczesnym podejściu do tworzenia aplikacji na Androida:



\* \*\*Język:\*\* \[Kotlin](https://kotlinlang.org/)

\* \*\*UI:\*\* \[Jetpack Compose](https://developer.android.com/jetbrains/compose) (Material Design 3)

\* \*\*Architektura:\*\* MVVM (Model-View-ViewModel) + Clean Architecture principles

\* \*\*Baza danych lokalna:\*\* Room Database (`AppDatabase`, `DAOs`)

\* \*\*Backend / Chmura:\*\* \[Supabase](https://supabase.com/) (widoczne w `SupabaseClient.kt`)

\* \*\*Wstrzykiwanie zależności:\*\* Manual DI / AppContainer (z pliku `di/AppContainer.kt`)

\* \*\*Asynchroniczność:\*\* Kotlin Coroutines \& Flow
